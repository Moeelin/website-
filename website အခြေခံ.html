Website တစ်ခုရေးသားရာတွင် HTML CODE ကို အများဆုံး အသုံးပြုသည်။

အစပြုရာတွင် ဖြင့် အစပြုရေးသားရသည်။ထို့နောက် htmlအဖွင့်နှင့်htmlအပိတ်ကိုရေးသားရသည်။၎င်းတို့ကြားတွင် codeများကိုရေးသားရသည်။codeအပိုင်းတွင် head နှင့် body ဟူ၍ ၂ပိုင်းရှိသည်။codeများရေးသားရာတွင် အဖွင့်တွင်< >ဖြင့်ရေးသား၍ အဆုံးတွင်ဖြင့် အဆုံးသတ်ပေးရသည်။head အပိုင်းတွင် unicode သုံး၍ရစေရန် meta charset ကိုခံရေးပေးရသည်။ (ဥပမာ -meta charset="UTF-8"/))
